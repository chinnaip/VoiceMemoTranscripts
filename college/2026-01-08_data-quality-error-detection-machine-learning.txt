Rule based and deterministic error detection is the research focus using classical, deterministic approaches in CAT pipelines. Core academic questions: What is the expressive power of rule based systems? Where do deterministic checks fail at scale? How do rule conflicts and coverage gaps arise? Sub themes include SQL based constraint enforcement, deterministic reconciliation logic, rule explosion, maintenance cost, and precision versus recall trade offs. The research gap is that rules detect known knowns but fail on novel error patterns, multi hop dependency failures, and partial or probabilistic mismatches. This becomes your baseline comparison.

Statistical and anomaly based error detection focuses on using statistical irregularities to flag potential CAT errors. Core academic questions: Can abnormal distributions indicate latent reporting errors? How do false positives impact regulatory workflows? Which features are stable across reporting days? Sub themes include distribution drift detection, price quantity latency, volume time based anomaly detection, Z score, IQR, and isolation forests. Limitations are that these approaches cannot explain why an error occurred and are sensitive to regime changes, volatility, and market events.

Machine learning for error classification involves learning to predict error types from historical CAT feedback. Core academic questions: Can CAT error labels be learned reliably? How do we handle extreme class imbalance? How transferrable are models across symbols and firms? Sub themes include supervised classifiers like XGBoost and MLP, feature engineering from event graphs, PU learning with positive unlabelled data, and weak supervision using rule outputs. Evaluation metrics include precision for compliance and time to detection with T plus 3 SLA impact, which transforms detection into a predictive task.

Graph based error propagation and dependency modelling focuses on modelling CAT data as event graphs instead of flat tables. Core academic questions: How do errors propagate across linked events? Can graph structure reveal root causes? Are certain nodes consistently causal? Sub themes include order lifecycle graphs where nodes equal events and edges equal dependencies, graph traversal for error impact analysis, graph neural networks for detection, and subgraph pattern mining. This is where your work becomes novel and publishable.
